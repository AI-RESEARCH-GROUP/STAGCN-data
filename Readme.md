# 暴力分拣数据集 #
本暴力分拣数据集主要包括：抛（falling）、踢（kicking）、正常（normal）、砸（smashing）、丢（throwing）五种基本暴力分拣动作。
## 文件结构 ##
```
| -- 暴力分拣数据集
     | -- skeleton.zip
     | -- violent_sorting.zip
```
其中` violent_sorting.zip `主要包含各种暴力分拣动作的原视频；\
` skeleton.zip `主要包含各种暴力分拣动作的骨骼数据。
